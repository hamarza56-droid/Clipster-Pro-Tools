System Overview

Clipster SaaS is a workflow automation platform that connects scraping, content filtering, reel processing and campaign submission into a single system.

🕷 SCRAPER ENGINE
Extract Instagram reels from target pages
Step 1 → Input pages
Step 2 → System fetches reels
Step 3 → Filters by duration & rules
Step 4 → Sends results to pipeline
🎯 CAMPAIGN SYSTEM
Define rules for each client campaign
• CS2Skin → logo filter ON
• Hellcase → gameplay only
• Tradeit → short reels priority
🎬 REEL PROCESSOR
Auto crop + background + branding system
Input reel → detect gameplay → crop → add logo → export
👥 TEAM SYSTEM
Track team performance and submissions
Members → tasks → earnings → campaign output tracking
